# Project 1: Fundamental Booster

Welcome to **Project 1: Fundamental Booster**!\
This project is a simple and interactive Python program designed to
collect basic personal information from the user and display it back in
a structured format.

------------------------------------------------------------------------

##  Features

-   Collects user's:
    -   Name\
    -   Age\
    -   Height\
    -   Favourite Number\
    -   Birth Year (approx.)
-   Displays collected data along with its **type** and **memory ID**
    (for learning purposes).
-   Beginner-friendly --- great for Python fundamentals practice.

------------------------------------------------------------------------

##  What You Learn

-   How to use `input()`\
-   Storing user data in variables\
-   Printing formatted output\
-   Understanding `type()` and `id()` functions\
-   Basic interactive Python program structure

------------------------------------------------------------------------

##  File Included

-   `prooject.1 fundamental booster.py` --- main Python script

------------------------------------------------------------------------

##  How to Run

1.  Make sure Python is installed.
2.  Open terminal/cmd.
3.  Run the script using:

``` bash
python "prooject.1 fundamental booster.py"
```
